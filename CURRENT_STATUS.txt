🎯 ЧТО НУЖНО СДЕЛАТЬ СЕЙЧАС:

┌─────────────────────────────────────────────────────────────────────────┐
│ 1. ПРОЧИТАЙТЕ ИНСТРУКЦИЮ                                                 │
└─────────────────────────────────────────────────────────────────────────┘

   cat XTREME_CASE_ONLY.txt

   Там пошаговая инструкция для настройки только XtremeCase

┌─────────────────────────────────────────────────────────────────────────┐
│ 2. ПОЛУЧИТЕ TELEGRAM API                                                 │
└─────────────────────────────────────────────────────────────────────────┘

   https://my.telegram.org/apps
   
   Получите:
   - api_id
   - api_hash

┌─────────────────────────────────────────────────────────────────────────┐
│ 3. НАСТРОЙТЕ .env                                                        │
└─────────────────────────────────────────────────────────────────────────┘

   cp .env.example .env
   nano .env

   Заполните обязательные параметры:
   - TELEGRAM_API_ID
   - TELEGRAM_API_HASH
   - TELEGRAM_PHONE
   - BOT_TOKEN
   - TARGET_CHANNEL_ID
   - XTREME_CASE_CHANNEL_ID
   
   И ВАЖНО:
   - ENABLE_XTREME_CASE=true
   - ENABLE_CIFROVOY_RAY=false  ← Отключен!

┌─────────────────────────────────────────────────────────────────────────┐
│ 4. УСТАНОВИТЕ ЗАВИСИМОСТИ                                                │
└─────────────────────────────────────────────────────────────────────────┘

   pip install -r requirements.txt

┌─────────────────────────────────────────────────────────────────────────┐
│ 5. ЗАПУСТИТЕ БОТА                                                        │
└─────────────────────────────────────────────────────────────────────────┘

   ./start_unified_bot.sh

   или:

   python unified_bot.py

🔄 КОГДА ПОЯВИТСЯ ДОСТУП К GOOGLE SHEETS:

1. Настройте Google Sheets API (см. NEW_ARCHITECTURE.md)
2. Скачайте credentials.json
3. В .env измените: ENABLE_CIFROVOY_RAY=true
4. Перезапустите бота

Тогда бот будет работать с ОБОИМИ источниками!
